- Config helps... -

"isWindows": This will let the PY OS Improved know your operating system's type
Boolean: "true", "false"

"cmd_theme": Default theme when you log in
Boolean: see "chthm" command with no argument

"enable_instant_show_time": Show the current time before username
Boolean: "true", "false"

"isUnregistered": you dont need to configure this...
Boolean: "true", "false"

"beep_when_finished": Beep everytime when you finished running a command
Boolean: "true", "false"

"auto_boot_choice": If you configured this the boot manager screen will skip and select your options
Boolean: "1" ~ "7"

"enablePassword": Enable password when you login(broken feature)
Boolean: "true", "false"

"show_password_when_typing": Enable if you want to see what you typed
Boolean: "true", "false"

"pwdstring": Password
Boolean: Custom

"allowShowNotify": Allow PY OS Improved send message to your desktop
Boolean: "true", "false"

"dualBoot": Enable dual boot(subsystem)
Boolean: "true", "false"

"dualBoot_startupCommand": Boot command of subsystem
Boolean: Custom

"dualBoot_OSName": What name will be showed in the boot manager?
Boolean: Custom

"venvEnable": Enable venv(Virtual Environment) if you need it
Boolean: "true", "false"

"venvPath": Where is your venv?
Boolean: Custom

"replace_python_command_to_python3": If you have compatibility problem, enable this
Boolean: "true", "false"

"disablePathShow": If path is too long in your shell, you can enable this
Boolean: "true", "false"

"shorter_welcome": If you think the welcome text too long, you can enable this
Boolean: "true", "false"

"faster_startup": Enable new version of the startup screen
Boolean: "true", "false"

"rsyscmd_when_cnf": Run command to host system, if the command is not found
